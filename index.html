A description of each domain is included below. Above are links to navigate through each activity, as well as which domain they are in.

DOMAIN SUMMARIES

DOMAIN 1:

In this domain we learned the basics of HTML as well as how programs typically work. We learned about scripts and comments, as well as how to organize our webpage into head and body tags. The head contains metadata and other details about the webpage, but the body has what the user is able to interact with. We also learned about noscript, which is the text that will show if the browser doesn't support what the website is running on.


DOMAIN 2:

In this domain, we learned the fundamentals of CSS and how it works. We learned about class selectors, classes, id, and id selectors. We also learned about internal CSS, inline CSS, and external CSS and when to use each. We should use external CSS to keep consistency, but internal CSS to style individual pages. Inline CSS is only for specific things.


DOMAIN 3:

In this domain we dived deeper into HTML and how specific tags can be used to enhance our webpage. We learned about tags such as footer tags, which are good for things such as company information, how to hide photo details and alt attributes which help when photos don't load. We also learned how to create forms and make them accessible using tabs and letters.


DOMAIN 4:

In this domain we dived deeper into HTML by attempting to use media in our webpages. We studied about tags such as control and video in order to implement videos and audio into our webpages, as well as let users interact with it. We learned a bit more about links as well, especially about images and their attributes.


DOMAIN 5:

In this domain we furthered our knowledge on CSS by learning about things pertaining to styling. We learned about absolute and relative positioning as well as how to center text, learning how to make our webpage both nicer and easier to comprehend in general. We learned more about grid columns and divs as well, and how their priorities work.


DOMAIN 6:

In this domain we learned how to make our pages more accessible and easy to comprehend. We learned how to change font sizes as well as about text hierarchies, as well as how to make our code easier to read and understand. We learned about organization and why it's important, in addition to understanding how browsers can run things differently and why it's important to take it into consideration.
